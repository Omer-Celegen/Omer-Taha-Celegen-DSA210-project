# Project Proposal: Impact of FED Interest Rate Decisions on Silver Prices (2021-2026)

## Motivation
[cite_start]The goal of this project is to analyze how Federal Reserve (FED) interest rate decisions affect silver ounce prices[cite: 25, 28]. [cite_start]Since silver is a key financial asset, understanding its correlation with interest rates is crucial for data-driven financial analysis[cite: 5, 10].

## Data Source & Collection
* [cite_start]**Silver Prices:** Historical daily data will be collected using the `investing.com`.
* [cite_start]**FED Rates:** Data will be enriched with official interest rate announcement dates from the FRED (Federal Reserve Economic Data)[cite: 18, 20].

## Data Characteristics
* [cite_start]**Time Frame:** Last 5 years (2021-2026)[cite: 26].
* [cite_start]**Sample Size:** Approximately 1,250 daily price points[cite: 26].

## Planned Analysis
* [cite_start]**EDA:** Visualizing price trends around FED meeting dates[cite: 30, 37].
* [cite_start]**Machine Learning:** Using regression models to analyze the impact of rate changes on silver price volatility[cite: 12, 38].
