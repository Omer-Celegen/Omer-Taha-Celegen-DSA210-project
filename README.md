# Project Proposal: Impact of FED Interest Rate Decisions on Silver Prices (2021-2026)

## Motivation
The goal of this project is to analyze how Federal Reserve (FED) interest rate decisions affect silver ounce prices. Since silver is a key financial asset, understanding its correlation with interest rates is crucial for data-driven financial analysis.

## Data Source & Collection
* **Silver Prices:** Historical daily data will be collected using the `investing.com`.
* **FED Rates:** Data will be enriched with official interest rate announcement dates from the FRED (Federal Reserve Economic Data) 'https://fred.stlouisfed.org/series/FEDFUNDS'.

## Data Characteristics
* **Time Frame:** Last 5 years (2021-2026).
* **Sample Size:** Approximately 1,250 daily price points.

## Planned Analysis
* **EDA:** Visualizing price trends around FED meeting dates.
* **Machine Learning:** Using regression models to analyze the impact of rate changes on silver price volatility.
